Place the four player portrait files in this folder.

The app maps avatars by player name and tries these filenames automatically:

- `aca.png`, `aca.jpg`, `aca.jpeg`, `aca.webp`
- `sveta.png`, `sveta.jpg`, `sveta.jpeg`, `sveta.webp`
- `peca.png`, `peca.jpg`, `peca.jpeg`, `peca.webp`
- `bucki.png`, `bucki.jpg`, `bucki.jpeg`, `bucki.webp`

If an avatar file is missing, the UI falls back to a styled initials badge.
